# subnyquist-spectrum-aoa
Sub-Nyquist wideband spectrum sensing and joint frequency-AOA estimation

## Stage 0 P0-3 result

The completed P0-3 search evaluates 1,662 sampling-rate combinations with
strict noiseless temporal and joint temporal-spatial candidate counts. Results
are in `results/p0_3_sampling_rate_search/`. Under the initial A-class screen,
no low-rate set reached both temporal ambiguity and at least 90% joint unique
coverage; the present decision is therefore **CONDITIONAL GO**. Spatial phase
does reduce candidates away from broadside, but it does not remove the severe
broadside limitation on the specified grid.
